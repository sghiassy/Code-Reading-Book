How to use this documentation

The DemoGL v1.3 documentation is split up in three parts: About DemoGL, Using DemoGL and DemoGL Reference.

For people new to DemoGL and effect programming it's wise to read first the documents in About DemoGL, then Using DemoGL and flip back and forth bewteen these two and the DemoGL Reference manual to get details about topics discussed in About DemoGL and Using DemoGL. The DemoGL Reference manual contains, besides reference documentation about the API also in depth information about technical aspects and script language syntax.

For people familiar with DemoGL v1.2 and earlier, it's wise to briefly browse throught the About DemoGL manual and the Getting started section in the Using DemoGL manual to pick up information about the new features and functionality in DemoGL v1.3 and then start reading the Advanced topics section in Using DemoGL to get confortable with the new features and how to embed DemoGL v1.3 in new applications and existing applications. Read the DemoGL Reference manual thouroughly to understand the differences between v1.2 and v1.3.

For all developers using DemoGL it's wise to know the OpenGL API as much as possible to be able to produce the best effects imaginable. For more information about programming effects using OpenGL, please visit one or more of the following sites:

  • ATi Technologies Developer site
  • nVidia Developer site
  • www.opengl.org, the main portal to everything OpenGL
  • Official OpenGL FAQ
