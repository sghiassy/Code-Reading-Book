DEMOGL_FileFree

DEMOGL_FileFree frees the memory pointed by the pointer passed as the parameter. Used to free up memory allocated by DemoGL, for example when loading a file using DEMOGL_FileLoad, due to the fact that the DemoGL DLL is another programmatic instance than the application calling DEMOGL_FileLoad and memory should be freed per instance; instances can't free memory allocated in other instances. 

void
DEMOGL_FileFree(
	const byte *pbyFile
);
Parameters.
pbyFile
Pointer to a block of memory allocated inside DemoGL.

Return values.

Remarks.
pbyFile typically points to a block data loaded with DEMOGL_FileLoad. Don't use DEMOGL_FileFree to free up memory not allocated inside DemoGL. Doing so can result in an application crash due to a GPF.

Requirements.
DemoGL v1.3 (build 0112 or higher)
